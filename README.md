# Repository for Dense Metric Depth Estimation via Event-based Differential Focus Volume Prompting (NeurIPS 2025)
Boyu Li, Peiqi Duan, Zhaojun Huang, Xinyu Zhou, Yifei Xia, and Boxin Shi, Dense Metric Depth Estimation via Event-based Differential Focus Volume Prompting, NeurIPS 2025.

[PDF](https://pub-e7bc7e5ed67b401a8e9e587c3dee22b6.r2.dev/publications/2025/Li_NeurIPS25.pdf)


Data are **coming soon**.


## Usage
1. Download pre-trained checkpoints from [Google Drive](https://drive.google.com/drive/folders/1vqGVEZZIj2S2YZgPc9xIH67iMkf05VHg?usp=drive_link).
2. Create an environment.
3. To evaluate, use:
`python test_fullnet.py [da/metric3d]`
To train the network, use:
`python train_fullnet.py`
